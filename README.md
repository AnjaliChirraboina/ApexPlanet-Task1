# ApexPlanet Internship - Task 1

## Project Title

Setting Up the Development Environment (PHP & MySQL)

## Description

This project demonstrates the successful setup of a local PHP development environment using XAMPP and the creation of a basic PHP application.

## Technologies Used

* PHP
* XAMPP
* Apache Server
* Git
* GitHub

## Project Structure

* `index.PHP` - Main PHP file that displays the output.
* `README.md` - Project documentation.

## How to Run the Project

1. Install XAMPP.
2. Start the Apache server.
3. Place the project inside the `htdocs` folder.
4. Open a web browser and visit:
   `http://localhost/MyFirstPHPProject/Task1`
